疑難排解

  • SSH
  • 用SSH連接遠端數據庫的時候會顯示”Permission denied (publickey).”
  • HTTPS
  • 在HTTPS不能clone
  • HTTPS執行push/pull時每次都需要密碼
  • SSH/HTTPS共同問題
  • Push後，無法將變更上傳到遠端數據庫
