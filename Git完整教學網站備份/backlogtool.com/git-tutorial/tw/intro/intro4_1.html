教學2 共享數據庫

在貝格樂上建立遠端數據庫

首先在貝格樂上建立Git遠端數據庫。

Tips

在貝格樂上，您的帳號需要具有管理者的權限才能建立Git數據庫。如果您的帳號沒有此權限，請委託有權限的用戶來建立數據庫。
貼心提示：您可以使用免費方案的貝格樂空間來建立數據庫喲。

從這裡註冊免費空間

請登錄貝格樂，從建立數據庫的專案選單裡點擊「Git」。如果空間裡還沒有建立任何專案的話請先建立專案。

若選單裡沒有顯示Git標簽，請為專案啟動Git功能，請在「專案設定」 > 「Git設定」內啟動Git功能。

啟動Git功能後，在Git設定畫面點擊「添加Repository」。

輸入欲新增數據庫的「Repository名稱」和「說明」後點擊「建立Repository」。這裡我們在名稱欄位輸入「tutorial」、説明則輸入「教學專用」。

如下圖所示，我們成功地添加了新的數據庫。

接下來Push到遠端數據庫吧！
