疑難排解

  • SSH
  • 用SSH連接遠端數據庫時會顯示”Permission denied (publickey).”
  • HTTPS
  • 不能clone
  • 執行push/pull時每次都需要密碼
  • SSH/HTTPS 共通
  • Push後，無法將變更上傳到遠端數據庫
